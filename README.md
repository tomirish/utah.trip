## Hike Details

### 🌊 Kanarra Falls — Day 2

| | |
|---|---|
| **Distance** | 3.7 miles out-and-back |
| **Difficulty** | Moderate |
| **Time** | 2.5–3 hours |
| **Permit** | $15/person · booked ✅ |
| **AllTrails** | alltrails.com/trail/us/utah/kanarra-falls |

Hike through Kanarra Creek in a narrow slot canyon with multiple waterfalls, ankle-to-knee deep water throughout. A 20-foot metal ladder gets you to the second waterfall — the turnaround point.

**Bring:** Waterproof shoes or neoprene socks · trekking poles · change of socks

---

### 🌅 Bryce Canyon Viewpoints — Day 2 (evening)

| Viewpoint | Notes |
|-----------|-------|
| **Sunset Point** | Thor's Hammer glowing red-orange · most iconic view |
| **Inspiration Point** | Widest panoramic of full amphitheater · fewer crowds |
| **Rainbow Point** | 9,115 ft · drive-through on way out Tuesday morning |

Aim to be at Sunset Point 45 minutes before sunset.

---

### 🥾 Peekaboo + Navajo Loop + Queen's Garden — Day 3

| | |
|---|---|
| **Distance** | 5.5–6 miles |
| **Difficulty** | Strenuous |
| **Time** | 3.5–4.5 hours |
| **Permit** | None |
| **AllTrails** | alltrails.com/trail/us/utah/navajo-loop-and-queens-garden-trail |

Start at Sunset Point. Navajo Loop descends through Wall Street (slot canyon, 20 ft wide, ancient ponderosa pines). Queen's Garden gets you up close to the Queen Victoria hoodoo. Peekaboo goes deepest into the amphitheater with far fewer crowds.

**⚠️ Start no later than 7 AM — zero shade, gets hot fast. Bryce sits at 8,000 ft — altitude makes climbs harder than expected.**

---

### 🐉 Belly of the Dragon — Day 3 (en route)

| | |
|---|---|
| **Distance** | 1.7 miles |
| **Difficulty** | Easy |
| **Time** | 20–30 minutes |
| **Cost** | Free |
| **AllTrails** | alltrails.com/trail/us/utah/the-belly-of-the-dragon |

A drainage tunnel under Hwy 89 where water erosion has sculpted the sandstone into a ribbed cave-like interior — completely dark in the middle. Right on the drive from Bryce to Zion.

**⚠️ Do not enter if raining — it floods fast.**

---

### 🏞️ Canyon Overlook Trail — Day 3 (afternoon)

| | |
|---|---|
| **Distance** | 1 mile out-and-back |
| **Difficulty** | Easy |
| **Time** | 45 minutes |
| **Permit** | None |
| **AllTrails** | alltrails.com/trail/us/utah/canyon-overlook-trail |

Best bang-for-buck hike in Zion. Starts at the east end of the Zion-Mt. Carmel Tunnel. Almost no elevation gain — weaves through slickrock to a dramatic panoramic of lower Zion Canyon and the Great Arch.

---

### 🌊 The Narrows (Bottom-Up) — Day 4

| | |
|---|---|
| **Distance** | Up to 9.4 miles (go as far as you want) |
| **Difficulty** | Moderate |
| **Time** | 3–5 hours |
| **Permit** | None for bottom-up day hike |
| **Gear** | Zion Outfitter · ~$45/person · neoprene socks + water shoes + pole |
| **Shuttle** | Temple of Sinawava (last stop) |
| **AllTrails** | alltrails.com/trail/us/utah/zion-narrows-trail |

The paved Riverside Walk (1 mile) leads to where the trail disappears into the river. From there you hike upstream through a slot canyon with walls up to 1,000 ft tall and 20–30 ft wide. Water is ankle-to-waist deep and cold. Go as far as you feel like.

**⚠️ Check NOAA before going. Flash floods arrive with no warning — do not enter if storms are forecast anywhere in the watershed.**

---

### 💎 Emerald Pools — Day 4 (afternoon)

| | |
|---|---|
| **Distance** | 3 miles |
| **Difficulty** | Moderate |
| **Time** | 2 hours |
| **Shuttle** | Zion Lodge (Stop 5) |
| **AllTrails** | alltrails.com/trail/us/utah/emerald-pools-trail |

Series of pools and small waterfalls in a shaded canyon alcove. The lower pool has a walk-behind waterfall that mists you — perfect in June heat. Upper pools are quieter.

---

### 🚲 E-Bike — Zion Canyon — Day 4 (afternoon)

| | |
|---|---|
| **Difficulty** | Easy — pedal-assist |
| **Route** | Pa'rus Trail + Zion Canyon Scenic Drive |
| **Book At** | Zion Cycles · zioncycles.com · 435-772-0400 |
| **Cost** | ~$100/bike · helmets + locks included |

Ride the same road as the park shuttle but at your own pace, stopping whenever you want. The Scenic Drive runs 6 miles through the heart of the canyon with 2,000-ft walls on both sides.

**⚠️ Only Class 1 pedal-assist e-bikes allowed in Zion NP (motor assists while pedaling, cuts off at 20 mph). Book ahead — June inventory is limited.**

## June Tips

| | |
|---|---|
| 🌅 **Start Early** | Every hike starts by 7 AM. Temps hit 90–95°F by midday. Most trails have zero shade. |
| 💧 **Water** | 2L per person minimum. Bring electrolyte tablets — water alone isn't enough on big days. |
| ☀️ **Sunscreen** | SPF 50. Reapply every 2 hours. High elevation + reflective red rock = brutal UV. |
| 📱 **Offline Maps** | Download AllTrails trails offline the night before — cell service in canyons is essentially zero. |
| ⚡ **Flash Floods** | Late June brings monsoon storms. Check NOAA before slot canyon hikes. Exit if weather builds. |
| 🚌 **Zion Shuttle** | Free, runs every 5–10 min to every trailhead. No personal vehicles in Zion Canyon. |
| 🧦 **Wet Feet** | Kanarra Falls + The Narrows = wet feet. Rent gear from Zion Outfitter (~$45/person). |
